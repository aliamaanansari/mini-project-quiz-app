Rules For Playing:-

1.Each level has 5 questions.
2.No points are deducted for wrong answers.
3.You get 30 seconds to answer each question.
4.Quitting the game will give you 0 score.
5.Point system.
  • Level EASY - 10 points for each question
  • Level MEDIUM - 15 points for each question
  • Level HARD - 20 points for each question
